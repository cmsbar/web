This is the real bar,
not a video.

A faithful, self-contained simulation of CMSBar editing a bakery's course page. Start a draft, then edit text, swap images from the media browser, embed a video, add rows to the info block, file an issue, and browse versions — every gesture is exactly what your editors get on the live site. (It runs entirely in your browser; no repo is touched.)

page-local editable shared (appears on many pages) press Esc to close panels

Tip: open Versions to see drafts as PRs, ⚑ Issues to file a note, and 🔗 Shared to light up content that appears on every page. Everything here is a front-end simulation — on a real site each Save is a commit and a pull request.
